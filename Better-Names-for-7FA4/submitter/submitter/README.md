## 现在是测试版

使用 Chrome 浏览器，打开[插件管理](chrome://extensions)，选择右上角的**开发者模式**，选择左上角的**加载已解压的扩展程序**，选择本项目所在的文件夹即可安装。

如果有后续更新，更新所在文件夹以后，在相应的插件点击刷新符号即可。

在 Chrome 浏览器右上角的插件按钮中，找到本插件，并固定到右上角。

先到已经登录的 7FA4 界面，选择**更新登录信息**。

然后到外站的**提交记录详情**页面，选择**发送提交记录**即可发送该提交信息到 OJ 的指定题目。

> 现在可以使用 vjudge 直接提交 洛谷/CF/AtCoder 的题目。
